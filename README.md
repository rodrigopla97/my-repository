# Hola, soy Rodrigo 👋

Este es mi portfolio personal. Lo construí para compartir un poco de quién soy, en qué trabajé y cómo contactarme. Si llegaste hasta acá, bienvenido.

## ¿Qué vas a encontrar?

- Una presentación de mi recorrido profesional
- Las tecnologías con las que trabajo día a día
- Un formulario para que a quie le interese se pueda contactar.

## Stack del proyecto

- **React 18** + **TypeScript**
- **Tailwind CSS** para los estilos
- **React Router v6** para la navegación
- **Vite** como bundler

## ¿Cómo lo levantás?

Necesitás tener Node.js 18 o superior.

```bash
npm install
npm start
```

Abre en [http://localhost:5173](http://localhost:5173)

```bash
# Para generar el build de producción
npm run build
```
